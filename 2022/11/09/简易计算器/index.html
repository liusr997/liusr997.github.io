简易计算器

只包括了加减乘除的运算

首先我们要从键盘获取数字和运算符，初始化

Scanner sc = new Scanner(System.in);
System.out.println("请输入首位数字");
double newNum;
newNum = sc.nextDouble();

然后我们要想的，当我输入第一位数字和第二位数字并进行运算后，得到的是第一对的运算结果

再次进行运算时，进行操作的数值为第二位数字和第一对的运算结果进行运算，以此类推，在进行第一次运算后，每当我们再次进行运算后。拿取的值都为上一次运算结果的第二位数字和上一次的运算结果

所以，为了获取每一次的运算结果值，我们要在main函数的首部创建一个动态数组，用来存储数值

为了获取动态数组的每一个索引，我们创建一个变量

ArrayList<Double> numArr = new ArrayList<>();
int count = 0;

接下来获取键盘中的运算符,考虑到可能进行多次运算，我们设置一个循环，并定义退出条件

System.out.println("按e退出运算");
    String cal;
while(true){
    System.out.println("请输入运算符");
    cal = sc.next();

}

获取参加运算的第二位数字,获取完对索引进行增加

double oldNum;
    System.out.println("请输入第二位数字");
    oldNum = sc.nexDouble();
    count++;

运算方法

   public static double add(double newNumber, double oldNumber) {
        return newNumber + oldNumber;
    }

    public static double sub(double newNumber, double oldNumber) {
        return newNumber - oldNumber;
    }

    public static double mul(double newNumber, double oldNumber) {
        return newNumber * oldNumber;
    }

除数不能为0,对此进行判断

注意，如果我将分子输入了0，数值会返回0

这样的话是不合法的运算，然而我们也会将count自增，从而导致动态数组的索引值发生错乱

所以我们要返回第二位数字输入的位置，增加个判断

 public static double dev(double newNumber, double oldNumber) {
        if (oldNumber != 0) {
            return newNumber / oldNumber;
        } else {
            return 0;
        }
    }

如果返回值为0，count不进行自增，并提示重输，直到运算为合法的

  while (true) {
                System.out.println("请输入数字");
                oldNum = sc.nextDouble();
                if (dev(newNum, oldNum) != 0) {
                    count++;
                    break;
                } else {
                    System.out.println("数值输入有误,请重新输入");
                }
            }

接下来判断符号并进行对结果的输出

if()判断是否为第一次进行运算

如果是将第一次的运算结果添加到动态数组中，方便之后运算的取值，此时的索引值为0

如果不是，要获取上一次的运算结果，也就是numArr[0]

由于我们每输入第二个数字后，count都会自增

所以我们第一次输出运算结果时的索引值为1，第二次及之后索引值必然时大于等于2 的

为了进行运算，要获取上一次结果的值，所以要使动态数组的索引值-2拿值

  switch (cal) {

可能多次进行运算并输出结果，所以我们从输入符号的位置添加个循环

最终程序

    public static void main(String[] args) {
        ArrayList<Double> numArr = new ArrayList<>();
        Scanner sc = new Scanner(System.in);
        int count = 0;
        double newNum;
        System.out.println("按e退出");
        System.out.println("请输入数字");
        newNum = sc.nextDouble();
        
        //多次进行运算
        while (true) {
            String cal;
            while (true) {
                System.out.println("请输入符号");
                cal = sc.next();
                if (cal.equals("e")) {
                    System.exit(0);
                } else if (!cal.equals("+") && !cal.equals("-") && !cal.equals("*") && !cal.equals("/")) {
                    System.out.println("符号输入有误，请重新输入");
                } else {
                    break;
                }
            }
            double oldNum;
            while (true) {
                System.out.println("请输入数字");
                oldNum = sc.nextDouble();
                if (dev(newNum, oldNum) != 0) {
                    count++;
                    break;
                } else {
                    System.out.println("有误");
                }
            }
            
            //运算并输出
            switch (cal) {
                case "+" -> {
                    if (count == 1) {
                        numArr.add(add(newNum, oldNum));
                        System.out.println("结果为" + add(newNum, oldNum));
                    } else {
                        numArr.add(add(numArr.get(count - 2), oldNum));
                        System.out.println("结果为" + add(numArr.get(count - 2), oldNum));
                    }
                }
                case "-" -> {
                    if (count == 1) {
                        numArr.add(sub(newNum, oldNum));
                        System.out.println("结果为" + sub(newNum, oldNum));
                    } else {
                        numArr.add(sub(numArr.get(count - 2), oldNum));
                        System.out.println("结果为" + sub(numArr.get(count - 2), oldNum));
                    }
                }
                case "*" -> {
                    if (count == 1) {
                        numArr.add(mul(newNum, oldNum));
                        System.out.println("结果为" + mul(newNum, oldNum));
                    } else {
                        numArr.add(mul(numArr.get(count - 2), oldNum));
                        System.out.println("结果为" + mul(numArr.get(count - 2), oldNum));
                    }
                }
                case "/" -> {
                    if (count == 1) {
                        numArr.add(dev(newNum, oldNum));
                        System.out.println("结果为" + dev(newNum, oldNum));
                    } else {
                        numArr.add(dev(numArr.get(count - 2), oldNum));
                        System.out.println("结果为" + dev(numArr.get(count - 2), oldNum));
                    }
                }
            }

        }
    }

    //运算方法
    public static double add(double newNumber, double oldNumber) {
        return newNumber + oldNumber;
    }

    public static double sub(double newNumber, double oldNumber) {
        return newNumber - oldNumber;
    }

    public static double mul(double newNumber, double oldNumber) {
        return newNumber * oldNumber;
    }

    public static double dev(double newNumber, double oldNumber) {
        if (oldNumber != 0) {
            return newNumber / oldNumber;
        } else {
            return 0;
        }
    }
